#### _The application will take in a an input number and will return the number typed out in words._

## Technologies Used

* _C#_
* _.Net_
* _Markdown_

## Description

_The application takes in a user input number and runs it through a function that splits the number to an array. The array is then passed into a branching statement that pushes the corresponding words into an output array_

## Setup/Installation Requirements


* _You can find the github repository [here]()_
* _Click the code button, and copy the https link_
* _In your in git bash or your preferred git terminal navigate to the directory you would like to store the project_
* _Enter: "git clone" followed by the https link_
* _Now that the repository is cloned to your computer, right click on the folder and click open with vs code_
* _Navigate into the NumbersToWords folder and type dotnet restore in the console to install dependencies_
* _You can now run the project by typing dotnet run in the console_
